Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de tornar o aprendizado de tecnologia mais envolvente e interativo. A proposta é unir conceitos de programação e desenvolvimento web com elementos de jogos, criando uma experiência educativa baseada em mecânicas de RPG.

No sistema, os alunos assumem o papel de aprendizes dentro de um universo fantástico, onde podem explorar diferentes ambientes como a taverna, o mapa de aventuras e o hall da fama. Cada área representa um espaço de aprendizado e progressão dentro da jornada.

As atividades são apresentadas como missões, onde os alunos precisam resolver desafios relacionados à tecnologia, lógica de programação e conceitos de informática. Ao concluir essas missões, os participantes evoluem dentro do jogo, ganhando experiência e desenvolvendo suas habilidades.

Entre os desafios presentes no projeto estão atividades interativas como caça-palavras sobre componentes de computador, desafios de lógica e exercícios relacionados ao desenvolvimento web.

O projeto foi desenvolvido utilizando tecnologias fundamentais da web, como HTML, CSS e JavaScript, com foco em uma arquitetura simples e didática, permitindo que os próprios alunos possam compreender a estrutura do sistema e evoluir o projeto ao longo do aprendizado.

Além de ser uma ferramenta educativa, o sistema também busca estimular o pensamento lógico, a criatividade e o interesse dos alunos pela área de tecnologia, transformando o processo de aprendizagem em uma verdadeira jornada de evolução.

Este projeto faz parte da minha atuação como Instrutor de Formação Profissional pelo SENAI, onde busco constantemente novas formas de tornar o ensino de tecnologia mais dinâmico, acessível e motivador para os alunos.
